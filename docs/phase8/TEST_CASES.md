# 测试用例

## 用户模块

- 首次打开 APP，完成 onboarding 后能进入首页
- 用户资料修改后能在用户页正确显示

## 饮食模块

- 输入高糖食物且带糖尿病标记时返回 `forbidden`
- 上传空文件名时仍能返回 mock image url

## 运动模块

- 手动补录 30 分钟高强度运动，积分应为 60
- 今日和本周 summary 接口能返回结构化结果

## 冥想模块

- 记录一次 10 分钟 session 后，今日汇总时长增加

## 睡眠模块

- 开始睡眠后状态应为 `recording`
- 结束睡眠后应返回评分、建议和事件列表

## 首页

- dashboard 接口返回总分、行动项和趋势数组
