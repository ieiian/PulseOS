# PulseOS 用户流程

## 1. 首次使用流程

```text
启动 App
  -> 欢迎页
  -> 基础资料输入
  -> 健康目标选择
  -> 权限引导
  -> 生成初始建议
  -> 进入首页
```

## 2. 饮食主流程

```text
首页/饮食页
  -> 点击拍照或快速记录
  -> 上传图片或输入食物
  -> 后端识别与规则判断
  -> 返回建议食用状态
  -> 展示详细说明
  -> 写入当日饮食记录
```

## 3. 运动主流程

```text
首次授权步数权限
  -> 自动拉取当日步数
  -> 计算心肺强化积分
  -> 展示今日完成度
  -> 用户可手动补录运动
  -> 更新日/周进度
```

## 4. 冥想主流程

```text
首页快捷入口/冥想页
  -> 选择呼吸模式
  -> 开始训练
  -> 动画与音频引导
  -> 结束训练
  -> 保存训练时长
```

## 5. 睡眠主流程

```text
睡眠页
  -> 点击开始睡眠
  -> 夜间录音
  -> 检测呼噜/梦话事件
  -> 点击结束睡眠
  -> 生成睡眠摘要
  -> 展示事件时间轴与建议
```

## 6. 用户模块流程

```text
用户页
  -> 查看资料摘要
  -> 编辑基础资料/目标
  -> 查看周/月统计
  -> 管理通知与权限
  -> 管理设备与账号
```

## 7. 首页闭环流程

```text
进入首页
  -> 查看今日总览
  -> 识别当前待完成事项
  -> 跳转到对应模块完成操作
  -> 返回首页看更新结果
```
