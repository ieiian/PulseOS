# Phase 5 冥想 MVP

## 完成内容

- 冥想数据模型
- 基础音频资源目录占位
- 呼吸训练节奏配置
- Android 冥想页面与播放控制骨架
- iOS 冥想页面与播放控制骨架
- 冥想记录上报接口
- 时长统计与展示

## 当前接口

- `POST /api/v1/meditation/sessions`
- `GET /api/v1/meditation/today`

## 当前呼吸模式

- `calm`: 4-2-6
- `focus`: 4-4-4
- `sleep`: 4-7-8

## 当前实现说明

- 音频播放控制目前为本地控制器骨架，不接真实媒体资源
- 资源目录已在 `assets/audio/` 预留
- 后端已支持记录 session 和返回今日汇总

## 下一步衔接

- Phase 6 睡眠可复用睡前呼吸模式和夜间氛围资源
- 后续可替换为真实播放器与音频文件
