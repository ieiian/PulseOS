# PulseOS UI 风格板

## 1. 视觉定位

- 关键词：专业、温和、清晰、可信、低压迫感
- 避免：医院管理系统感、纯工具感、社交化喧闹感

## 2. 主题方向

### 全局基调

- 主基调：蓝绿中性色
- 页面背景：浅暖灰或低饱和浅渐变
- 卡片：白色或极浅灰，轻阴影

### 模块主题色

- 用户：石墨灰 / 蓝灰
- 饮食：草木绿 / 柔和橙
- 运动：活力橙 / 珊瑚红
- 冥想：雾蓝 / 青绿
- 睡眠：深蓝 / 静谧靛蓝

### 语义色

- 成功：柔和绿色
- 提醒：琥珀黄
- 风险：暖红
- 禁止：深红

## 3. 组件风格

- 卡片圆角：16-20
- 主按钮：实心高对比
- 次按钮：浅底或描边
- 图标：线性或半填充统一
- 图表：结论优先，辅助线弱化

## 4. 首页信息层级

1. 今日概览
2. 今日总分
3. 当前最重要任务
4. 模块摘要卡
5. AI 建议
6. 趋势摘要

## 5. 用户页信息层级

1. 用户信息卡
2. 健康目标卡
3. 关键身体指标
4. 周/月统计入口
5. 设置与权限
6. 设备与账号管理

## 6. 动效边界

- 首页卡片渐入
- 呼吸训练节奏动画
- 积分增长反馈
- 睡眠开始/结束过渡

避免：

- 高频闪烁
- 大装饰动画
- 与健康数据无关的花哨转场

## 7. 适配原则

- 单手操作优先
- 夜间睡眠场景降低刺激
- 大字与高对比模式预留
